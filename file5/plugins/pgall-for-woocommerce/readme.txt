== Description ==

워드프레스 쇼핑몰 전문 코드엠샵에서 제공하는  우커머스 결제 플러그인 ‘심플페이’는 쇼핑몰에 대한 개발 지식이 없어도 누구나 1분안에 결제 플러그인 설치 및 연동, 설정, 업데이트가 쉽고 간편하도록 제작된 대한민국 온라인 결제 시스템 입니다.

대한민국에서 우커머스를 가장 많이 경험하고, 가장 많은 노하우를 가지고 있는 코드엠샵에서는 심플페이 워드프레스 결제 플러그인에 결제 기능을 비롯한 우커머스 쇼핑몰 운영에 필요한 다양한 기능을 무료로 제공하며, 보다 효율적이게 쇼핑몰 운영을 할 수 있습니다.

[결제 기능]
- 일반결제, 간편결제, 면과세 복합과세, 수기결제, 정기결제 (Woocommerce Subscription)

- 숏코드 결제 : 상품 페이지에서 결제가 아닌 숏코드 결제로 블로그 및 일반 페이지에서도 결제 지원
- PC, 모바일, 하이브리드앱 결제 지원
- 확장형 : 예약결제, 후불결제, 선결제 지원

[운영 기능]
- 매출통계 : 우커머스 주문상태에 따른 일간, 주간, 월간, 년간, 요일, 시간대별 매출통계
- 결제통계 : 결제 수단별 분석 / PC, 모바일에 대한 디바이스 분석 / 결제 시도, 성공, 포기, 실패건에 대한 분석
- 상품별, 카테고리별 주문상태 설정 : 결제 완료 시 주문상태 설정 및 주문처리 자동변경 설정
- 결제 페이지 내 사전 리뷰 작성, 구매평, 별점 자동 등록
- 언어별 결제수단 노출제어
- 결제 영수증, 현금영수증 발급
- 전체환불, 부분환불
- 확장형 : 심플페이 DIY 체크아웃 - 결제 페이지 (체크아웃 페이지) 커스터마이징 기능

[커스텀 기능]
- 결제창 언어설정 (한국어, 영어, 일어, 중국어)
- 테마에 어울리는 결제창 색상 변경 및 자사몰 로고 삽입
- 결제 로딩 이미지 변경

== 플러그인 설치안내 ==
1. 워드프레스 플러그인 메뉴에서 코드엠샵, 심플페이, 워드프레스 결제 등을 검색하여 설치 또는 다운로드 받은 파일을 wp-content/plugins/pgall-for-woocommerce 디렉토리에 업로드하여 설치합니다.
2. 심플페이 (구. PGALL  결제관리) 관리 메뉴에서 이용할 결제 대행사 선택 후 상점 아이디 설정 후 카드사 심사를 진행합니다.

== Frequently Asked Questions ==

= 결제 플러그인에서 여러 결제 대행사 결제를 이용할 수 있나요? =
심플페이 결제 플러그인 하나로 다양한 결제 대행사의 여러 결제 서비스를 동시에 이용할 수 있습니다.
코드엠샵의 심플페이는 복수 PG 이용 시 별도의 비용이 추가 되지 않습니다.


= 현재 타사의 결제 플러그인을 이용하고 있는데, 심플페이 교체 사용이 가능한가요? =
심플페이에서 제공하는 결제 대행사를 이용하고 계신 경우 상점 아이디 추가 발급을 통해 별도 카드사 심사없이 심플페이 이용이 가능합니다.

결제 플러그인 교체 설정은 매뉴얼 사이트를 확인 해 주세요. <a href="https://bit.ly/pg-change/">https://bit.ly/pg-change</a>
