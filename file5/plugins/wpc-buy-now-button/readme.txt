WPC Buy Now Button is the ultimate time-saving plugin that helps customers skip the cart page and get redirected right straight to the checkout step.

== Description ==

In modern daily life, saving time is the key element to business success. WPC Buy Now Button for WooCommerce is the ultimate time-saving plugin that helps customers skip the cart page and get redirected right straight to the checkout step. The Buy Now button can easily be added at any place you’d like to assist buyers during their purchasing process on the single product page or in the shop archive. Increase sales and reduce the bounce rate with a faster and customer-friendly checkout for online stores.

WPC Buy Now Button also has a feature to reset the cart before redirecting buyers to the checkout page to make sure the cart is properly updated. Users are able to change the wording of the button to get a better call for action effect or set up the parameter for the button as well.

Simple to use, just one click installation then you’ll have a nice Buy Now button in your shop.

= Live demo =

Visit our [live demo](https://demo.wpclever.net/wpcstore/ "live demo") here to see how this plugin works.

= Key Features =

- Change the position on the single product pages or shop archive
- Change the text for the Buy Now button
- Change the parameter for the button
- Enable/disable cart resetting before Buy Now event
- Choose the redirect page URL
- Compatible with most WordPress themes, WPC, and WooCommerce plugins

= Widely compatible with WPC plugins =

WPC Buy Now Button for WooCommerce can work with:

- Simple and variable products
- [WPC Product Bundles for WooCommerce](https://wordpress.org/plugins/woo-product-bundle/)
- [WPC Composite Products for WooCommerce](https://wordpress.org/plugins/wpc-composite-products/)
- [WPC Frequently Bought Together for WooCommerce](https://wordpress.org/plugins/woo-bought-together/)
- [WPC Grouped Products for WooCommerce](https://wordpress.org/plugins/wpc-grouped-product/)
- [WPC Force Sells for WooCommerce](https://wordpress.org/plugins/wpc-force-sells/)
- [WPC Smart Quick View for WooCommerce](https://wordpress.org/plugins/woo-smart-quick-view/)
- Most WordPress themes and WooCommerce plugins

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Buy Now Button", Install & Activate it
4. Go to WP-admin > WPClever > Buy Now Button to configure
